### 用法
1.项目下载到本地
2.在 Chrome 浏览器中打开 "chrome://extensions/" 页面，在页面上方打开 "开发者模式"，然后点击 "加载已解压的扩展程序" 按钮，选择我们项目的根目录。这样就可以安装插件成功了。
## 测试网站地址：
 https://x.com/search?q=hah&src=typed_query&f=top
### 功能介绍(注意一定是英文网站)
#### 自动检索匹配"@XXX","#XXX","$XXX"，插入到该币对值前方，并且绑定了点击事件.
